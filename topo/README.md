- ONT: 光猫
- routerB/routerC: 目前是配置静态 IP，关闭 DHCP，当个 AP
- Z0: 危险区，VLAN 10，仅 IPv4，局域网内互相隔离，禁止直接访问路由器或者光猫
  - TV: 我很希望让它能访问 NAS 或者可以连手机、游戏机，但还是觉得危险，移到 Z0
- Z1: 访客区，VLAN 20，仅 IPv4，VLAN 局域网内可以互相访问，禁止直接访问路由器或者光猫
- Z2: 信任区，VLAN 30，IPv4+IPv6，不隔离，可以直接访问路由器或者光猫
  - VPN: 通过公网 IPv6 对外提供服务，包括但不限于 wireguard/tailscale/openvpn
